# 프로젝트명: my_game

### 소개

- React 기반 게임 만들기
<br/>

### 게임 목록 및 설명

<details>
  <summary><strong>화살표 게임</strong></summary>

  ### 설명
  - 상자의 색상에 맞는 커맨드를 입력해 점수를 획득하는 간단한 게임입니다.
  <br/>
  
  ### 구현 기능
  - **기본 게임 기능**
      - 커맨드로 상자 맞추기
      - 피버타임
  - **게임 중지, 재시작**
  - **게임 종료 후 결과 확인**
  - **게임 설정 변경 및 리셋**
       - 시간 변경
       - 커맨드 변경
